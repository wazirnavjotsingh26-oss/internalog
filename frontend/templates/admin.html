Data Pipeline Overview

1
OpenStreetMap (Overpass API)

Fetches all nodes/ways/relations tagged as cemeteries within the state bounding box.

→
2
Nominatim Geocoding

Reverse geocodes lat/lon to fill missing city, county, and ZIP code fields.

→
3
Google Places API

Enriches with phone, website, and opening hours. Validates match by name similarity and distance.

→
4
MongoDB Atlas

Upserts records — prevents duplicates using name + coordinates as unique key.
